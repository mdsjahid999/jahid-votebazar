
# জাহিদ ভোটবাজার (Jahid VoteBazar) - হোস্টিং নির্দেশিকা

এই প্রোজেক্টটি ইন্টারনেটে লাইভ করার জন্য নিচের সহজ ধাপগুলো অনুসরণ করুন:

## ১. সবচেয়ে সহজ উপায় (Vercel ব্যবহার করে)
১. আপনার কোডগুলো [GitHub](https://github.com/)-এ একটি রিপোজিটরিতে আপলোড করুন।
২. [Vercel](https://vercel.com/)-এ যান এবং GitHub দিয়ে লগইন করুন।
৩. আপনার প্রোজেক্টটি Import করুন এবং **Deploy** বাটনে ক্লিক করুন।

## ২. কমান্ড লাইনের মাধ্যমে (বিকল্প উপায়)
আপনার কম্পিউটারে Node.js ইন্সটল করা থাকলে নিচের কমান্ডগুলো রান করতে পারেন:

```bash
# ডিপেন্ডেন্সি ইন্সটল করতে
npm install

# প্রোজেক্টটি বিল্ড করতে
npm run build
```

বিল্ড করার পর তৈরি হওয়া `dist` ফোল্ডারটি আপনি [Netlify Drop](https://app.netlify.com/drop)-এ ড্র্যাগ অ্যান্ড ড্রপ করে সরাসরি হোস্ট করতে পারেন।

## ৩. সুপাবেস (Supabase) সতর্কতা
আপনার প্রোজেক্টে সুপাবেস কানেক্ট করা আছে। হোস্টিং করার পর সুপাবেস ড্যাশবোর্ডে গিয়ে আপনার নতুন ওয়েবসাইটের লিঙ্কটি "Authentication -> URL Configuration" এ `Site URL` হিসেবে সেট করে দিলে লগইন সিস্টেম একদম নিখুঁতভাবে কাজ করবে।
